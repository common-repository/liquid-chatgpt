== 3rd Party Service ==
We are using Unsplash as a third-party API query. Keep in mind that the images you use can be used completely according to Unsplash's privacy policy. Also, keep in mind that you cannot query Unsplash without your Unsplash API key.

Only a "search term" related to the relevant article is made to Unsplash and the results are displayed.

For more details, please review here: https://unsplash.com/privacy 


== Installation ==
1. Install the plugin through the Wordpress plugin repository or by uploading the ZIP file to your Wordpress site.
2. Activate the plugin through the 'Plugins' menu in Wordpress.
3. Access the plugin settings page and configure to your preference.

== Frequently Asked Questions ==
= How do I use AI Engine for WordPress? =
To use AI Engine for WordPress, simply add your OpenAI API key, which can be found at https://platform.openai.com/account/api-keys.

= How do I obtain Featured Images for my articles? =
To obtain Featured Images for your articles, you need to add an Unsplash API key, which can be found at https://unsplash.com/oauth/applications.

= How does AI Engine for WordPress work? =
Liquid AI uses OpenAI technology to generate content for your Wordpress site. Simply input your topic and desired word count, and the plugin will generate high-quality content for you in a matter of minutes.

= Is AI Engine for WordPress easy to use? =
Yes, AI Engine for WordPress has a user-friendly interface that makes it easy for you to use the plugin with no prior AI experience required.

= What kind of content can I generate with AI Engine for WordPress? =
You can generate articles and featured image with AI Engine for WordPress.
